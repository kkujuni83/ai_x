ch05_함수 요약정리

파이썬 함수 요약

- 정의: def 함수명(매개변수1, ...): 로 시작, 코드 블록 들여쓰기.

- 호출: 함수명(인수1, ...) 형태로 실행, 결과 반환 가능 (return).

- 매개변수: 필수, 기본값, *args (가변 위치), **kwargs (가변 키워드).

- 인수: 위치 인수, 키워드 인수 순서 중요.

- 스코프: 지역 (함수 내부), 전역 (함수 외부 global 키워드 사용).

- 람다 함수: lambda 인자: 표현식 익명 함수, 간단한 기능에 사용.

- 내장 함수 (예시):
변환: int(), str(), list(), tuple().
수학: abs(), sum(), max(), min().
순회: len(), range(), enumerate().
입출력: print(), input().
기타: type(), help().

- 장점: 코드 재사용, 가독성 향상, 유지보수 용이.

- 주의: 함수 중복 정의 시 마지막 정의로 덮어쓰기 됨.
