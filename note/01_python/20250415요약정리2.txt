ch05_함수 요약정리

- 코드 묶음 : 특정 작업을 수행하는 명령어들을 하나로 묶어 놓은 것.
- 재사용 가능 : 한 번 만들어 놓으면 필요할 때마다 여러 번 호출해서 사용 가능.
- 모듈화 : 코드를 논리적인 단위로 나눠서 관리하기 쉽고, 전체 프로그램 구조를 보기 편하게 만듦.

- 함수를 수행하면 갔다가 다시 돌아오는 것을 함.
- 함수형 언어는 파이썬과 자바스크립트만 가능

print : 기본함수 / 파이썬 기본함수(import 하지 않고 사용할 수 있는 함수)

내장 함수 (Built-in Functions) 기초
print(), len(), type(), int(), float(), str(), bool(), input(), range() 등.


- 코드 중복 방지 : 똑같은 코드를 여러 번 쓸 필요 없이 함수 호출 한 번으로 해결.
- 가독성 향상 : 복잡한 코드를 함수로 감싸서 의미 있는 이름으로 표현하면 이해하기 쉬움.
- 유지보수 용이 : 기능별로 분리되어 있어 수정해야 할 부분을 쉽게 찾고 관리 가능.
