▣ 파일 입출력(File I/O) 프로그래밍 요약

- 파일 입출력은 프로그램이 파일을 읽거나 쓰는 기능

- 주로 open() 함수를 사용해 파일을 열고, 작업 후에는 close()로 닫음.

- 'r' 모드는 읽기, 'w'는 쓰기, 'a'는 추가(append)를 의미

- 파일에서 읽을 때는 read(), readline(), readlines()를 사용

- 파일에 쓸 때는 write() 또는 writelines()를 사용

- with open(...) as f: 구조를 쓰면 자동으로 파일을 닫아주기 때문에 안전

- 텍스트 파일 외에도 바이너리 파일('rb', 'wb')도 다룰 수 있음.

- 예외 처리(try-except)를 통해 파일이 없거나 읽기 오류에 대비가능.

- 데이터 저장, 로그 기록, 설정 관리 등에 자주 사용

- 한글 인코딩 문제를 피하려면 encoding='utf-8'을 명시하는 것이 좋음.
